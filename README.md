**Mobile GaoOS** is a superset on top of [NixOS Linux](https://nixos.org/nixos/),
[Nixpkgs](https://nixos.org/nixpkgs/) and [Nix](https://nixos.org/nix/),
aiming to abstract away the differences between *mobile* devices.

In four words: *"NixOS, on your phone"*.

Read more: [About Mobile GaoOS](doc/about.md).

> **Note**: Mobile GaoOS is only expected to build successfully against the **unstable** branch of Nixpkgs.

## Documentation

* [The Mobile GaoOS website](https://Gao-OS.github.io/Mobile-GaoOS/) hosts the rendered [doc folder](https://github.com/Gao-OS/Mobile-GaoOS/tree/development/doc) for Mobile GaoOS documentation.

As Mobile GaoOS is a superset of NixOS Linux, the following documentation applies:

* [NixOS Manual](https://nixos.org/nixos/manual) - how to install, configure, and maintain a purely-functional Linux distribution
* [Nixpkgs Manual](https://nixos.org/nixpkgs/manual/) - contributing to Nixpkgs and using programming-language-specific Nix expressions
* [Nix Package Manager Manual](https://nixos.org/nix/manual) - how to write Nix expressions (programs), and how to use Nix command line tools

## Continuous Integration

There is no published artifacts for the time being, but the documentation
should point you towards resources that you can use to get started quicker when
your device is supported.

## Contributing

There are two ways your contributions can reach this project.

* [Contributing to Mobile GaoOS](CONTRIBUTING.md)
* [Contributing to Nixpkgs or NixOS](https://github.com/NixOS/nixpkgs/blob/master/CONTRIBUTING.md)

Contributions are always welcome. If it's your first time contributing:
jump in and don't be scared. Mistakes in your first contributions are room for
improvement; it is not in our nature to scold new contributors.
